# VC-ME
Videocall web application using webRTC and nodeJS
<h1>Running Application</h1>
<li>npm ci</li>
<li>node app.js</li>
<br>
<h1>Features</h1>
<li>Screen Sharing</li>
<li>Screen Recording</li>
<li>Real Time Chat</li>
<li>Video and Audio stream toggle</li>
<li>Mutiple Participants</li>
<li>Full Screen</li>
